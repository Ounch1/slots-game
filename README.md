## Tests
The project contains unit, integration, and end-to-end (E2E) tests.
Follow these commands to run them:
#### Start the Application
```npm start```
#### Run Unit & Integration Tests
```npm test``` 
#### Run End-to-End (E2E) Tests
```npm run test:e2e```
